Linux-Cheatsheet

1) Process management command:

  1.  ps  :-  Show process status. Often used with grep e.g.,  ps-aux

  2. top :-  Display sorted information about processes

  3. htop :- Display sorted information about processes with visual highlights. It allows you to scroll vertically and horizontally, so you can see every process running on your system and commands.

  4. kill PID :- Kill a process specified by its process ID PID, which you obtain using the ps command.

  5. pgrep :- This command used to search process by name and return their process ID (PID).  e.g., pgrep <name>S earch PID by Name
