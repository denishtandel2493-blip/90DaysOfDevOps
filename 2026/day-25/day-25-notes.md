## Git Reset 
          git reset is used to: Undo commits, Unstage files, Move branch pointer.
          3 Types of Git Reset: --soft Reset,  --mixed Reset,  --hard Reset
          --soft Reset :-
                Undo commit, but keep everything ready to commit again.
          --mixed Reset :-
                Removes files from staging area, Undo commit and unstage files.
          --hard Reset :-
                Delete everything and go back.

## Git Revert
      git revert is used to undo a commit by creating a new commit that reverses the changes.
      It does NOT delete history, It is safe for shared / pushed branches
