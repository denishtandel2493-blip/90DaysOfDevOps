## Lists
    There are two ways to write a list in YAML:  Block style list (using -)
        hobbies:
          - coding
          - reading
          - cricket
      Inline style list (using [ ])
          hobbies: [coding, reading, cricket]
      Both represent the same data.

## Multi-line Strings
    Difference between | and >:
    | (pipe style), Preserves line breaks exactly, Best for scripts, commands, configuration blocks

    > (folded style),  Converts newlines into spaces, Best for long descriptions or paragraphs
